Collapse / Expand

  • Click toggle → sidebar collapses to icons-only (72px)
  • Page content shifts left when collapsed
  • Reload page → collapsed state is remembered
  • Icons remain centered when collapsed
  • Click toggle again → sidebar expands back to 260px

Active Page Highlighting

  • Current URL path matches a link → it's highlighted in blue
  • No other links are highlighted at the same time
  • Test by adding ?path=/app/dashboard to URL and refreshing

PRO / NEXA Gating

  • "NEXA Exclusive" section is hidden by default

  • Add body.nexa-user class in DevTools → NEXA section appears

  • Content fills full width on mobile

Console / Errors

  • Open DevTools → Console tab
  • No JavaScript errors on load
  • No "No mount point found" warning (means div was found)
  • CSS was injected into <head> (check Elements tab)

Deployment Steps

Step 1 — Deploy the Worker

  1. Make sure Wrangler is installed: npm install -g wrangler
  2. Log in: wrangler login
  3. Deploy from the repo root:
    wrangler deploy workers/mtg-broker-sidebar.js --name mtg-broker-sidebar --compatibility-date 2024-01-01
  4. Copy the Worker URL from the output (e.g. https://mtg-broker-sidebar.YOUR-SUBDOMAIN.workers.dev)
  5. Test it: visit https://mtg-broker-sidebar.YOUR-SUBDOMAIN.workers.dev/sidebar.js — you should see JavaScript

Step 2 — Update Webflow (replaces the giant embed)

  1. In Webflow Designer, go to each app page that has the sidebar embed
  2. Delete the giant sidebar embed (the one with all the HTML/CSS/JS)
  3. Add a new Embed element where the sidebar was, with just:
    <div id="mtg-sidebar"></div>
  4. In Site Settings → Footer Code, add once (applies to all pages):
    <script src="https://mtg-broker-sidebar.YOUR-SUBDOMAIN.workers.dev/sidebar.js"></script>
  5. Publish the Webflow site and verify on a live app page

Step 3 — Update the AI Loan Finder React App

  1. Open apps/ai-loan-finder/index.html
  2. Inside <body>, add before the <div id="root">:
    <div id="mtg-sidebar"></div>
  3. At the bottom of <body>, add:
    <script src="https://mtg-broker-sidebar.YOUR-SUBDOMAIN.workers.dev/sidebar.js"></script>
  4. Make sure the app's CSS sets --navbar-height and pushes content right of the sidebar:
    margin-left: 260px; (or use var(--sidebar-width))
  5. Rebuild and deploy: cd apps/ai-loan-finder && npm run build

Future Sidebar Updates (the big win)

  1. Edit workers/mtg-broker-sidebar.js — change CSS, HTML, or links
  2. Run: wrangler deploy workers/mtg-broker-sidebar.js --name mtg-broker-sidebar --compatibility-date 2024-01-01
  3. Done — ALL pages (Webflow + React app) pick up the change automatically
  4. No Webflow Designer changes needed, no app rebuild needed
